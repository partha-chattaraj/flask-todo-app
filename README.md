# Flask To-Do App 📝

A simple To-Do web application built using Python Flask.

## 🚀 Features
- Add tasks
- Delete tasks
- Simple user interface
- Dynamic task list

## 🛠️ Tech Stack
- Python
- Flask
- HTML

## ▶️ How to Run
1. Clone the repository
2. Open project folder
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
